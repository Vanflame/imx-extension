Install instructions (Chrome / Edge)

1
Open chrome://extensions (or edge://extensions) and toggle Developer mode ON.
2
Drag and drop the downloaded .crx file into the extensions page.
3
Click the puzzle icon and Pin “Immutable Stats Checker” to the toolbar.

On Android (Kiwi Browser): open chrome://extensions → Developer mode → Load unpacked → select the extracted folder.

What it does

  • Captures your Bearer token used by Immutable Play.
  • Shows tier, % progress, weekly points, eligibility, and quests.

Trouble installing? Try restarting the browser after enabling Developer mode, or use the unpacked method if .crx install is blocked.
